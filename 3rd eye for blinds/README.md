# THIRD EYE FOR BLINDS

 ***TO MEASURE DISTANCE AND GIVE A BEEP SOUND ACCORDING TO THE DISTANCE*** 

**REQUIRED COMPONENTS-**

     1. Arduino UNO

     2. Software- Arduino IDE

     3. jumper wires (male-male,male-female)

     4. HC-SR04 ultra sonic sensor

     5. Buzzer
     
     6. hand glove
